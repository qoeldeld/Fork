울트라 미팅 - 요구사항 명세서

1. 개요
그룹간의 회의에 있어서 적당한 장소를 찾기가 번거롭고 회의 장소 전달에서의 문제점을 보완하기 위해 고안된 안드로이드 기반 어플로 회의 참여자들이 본인의 위치와 회의 장소를 한눈에 파악할수 있도록 한다.




1.1 목표
(1) 정확한 메뉴를 쉽게 활용할 수 있도록 단순화하여 신뢰성 있는 시스템이 되도록 한다.
(2) 원하는 장소를 신속하게 찾아 볼 수 있도록 한다.
(3) 회의에 적합한 오프라인 모임 장소 제공으로 회의 효율성 상승
(4) 프로젝트를 통해 구성원간의 협동심과 단결력을 배양한다.

2. 기능 요구사항

.사용자에게 보여지는 지도 구현( Google Map연동)
.위치서비스를 이용한 회의 참여자의 위치 확인 기능
.오프라인 회의에 적합한 장소 지정 및 추천 기능


2.1 유저 인터페이스 요구사항
(1) 로그인 화면
- 어플리케이션을 실행하면 첫 화면으로 로그인 버튼이 화면에 나타난다.
- 로그인 버튼을 누르면 지도화면이 나타난다.
(2) 메인 화면(지도)
- 로그인이 되면 메인 화면이 나타난다.
- 하단에 '현재위치','친구목록','그룹','화의장소'의 메뉴가 나타난다.
(3) 현재위치 버튼
- 메인화면에서 '현재위치' 버튼을 눌렀을 시 현재위치로 복귀.
(4) 친구목록 버튼
-  메인화면에서 '친구목록' 버튼을 누르면 친구목록 창이 나타난다.
- 친구 목록 창에서 등록된 친구를 선택하면 친구의 위치가 나타난다.
- 상단 오른쪽 '친구 추가' 버튼을 누르면 아이디를 입력해 친구를 추가 할 수 있다.
(4) 그룹 버튼
- 메인화면에서 '그룹'버튼을 누르면 그룹 목록이 나타난다.
- 오른쪽 상단에 '그룹 생성' 버튼을 누르면 친구목록에 있는 친구들을 여러명 선택해 그룹으로 만들수 있다.
(5) 회의장소 버튼
- 메인화면에서 '회의장소' 버튼을 누르면 회의장소 목록이 나열된다.
-  장소를 선택하면 회의를 하게될 그룹을 선택하는 창이 나타난다.
- 그룹을 선택하면 구성원들에게 회의장소가 공유된다.


2.2 기능적 요구사항
(1) 로그인 기능
- 페이스북 아이디로 로그인 할 수 있다.
- 페이스북에서 제공하는 개발 툴킷을 사용하여 구현한다.
(2) Google Map 연동 
- 현재 위치와 구성원의 위치와 회의장소를 파악할 수 있다.
(3) 친구, 그룹, 회의장소 목록 
- 각 계정별로 친구, 그룹, 회의장소의 각각의 목록이 유지되어야 한다.
